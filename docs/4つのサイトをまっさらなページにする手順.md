# 4つのサイトをまっさらなページにする手順

**ultra.omamori / snobbycover / hickeytattoo / gummygummy** の 4 つのサブドメインで、いまは「まっさらなページ」が表示されるようにしてあります。

---

## いまの動き

- **home.whenpigsfly.jp** … これまでどおり **Wii メニュー** が表示される
- **gummygummy.whenpigsfly.jp** … **STORY_GUM**（Shopify の gummybond 動画・画像ページ）を表示
- **hickeytattoo.whenpigsfly.jp** … **STORY_KISS**（Shopify の KISS 動画・画像ページ）を表示
- **snobbycover.whenpigsfly.jp** … Snobby Book Cover の専用ページ
- **ultra.omamori.whenpigsfly.jp** … **まっさらなページ**（薄いグレー背景だけ）

同じ Vercel プロジェクト（同じ `out`）のまま、**開いたドメイン** で出し分けています。

### STORY_GUM / STORY_KISS のメディアファイル

動画・画像は `public/gummygummy/` と `public/hickeytattoo/` に配置してください。

- **gummygummy**: `gummybond_film01_web.mov`, `gummybond_film02_web.mov`, `gummybond_film03_web.mov`, `38.png`, `39.png`, `40.png`, `gum2.png`
- **hickeytattoo**: 同様のファイル（KISS 用に差し替え可能）

Shopify CDN の URL を使う場合は `app/data/shopifyStory.ts` の `SHOPIFY_CDN` を設定してください。

---

## まっさらなページを変えたいとき（手順）

### 1. どのファイルを触るか

**編集するファイル:** `app/components/BlankPage.tsx`

このコンポーネントが、4 つのサブドメインで表示される「まっさらなページ」の中身です。

### 2. やること

1. **`app/components/BlankPage.tsx`** をエディタで開く  
2. 中身を好きなデザインに書き換える  
   - いまは「薄いグレー背景だけ」の `div` 1 つ  
   - テキスト・画像・リンク・レイアウトなどを追加して OK  
3. 保存する  

### 3. 本番に反映する

```bash
npm run build
git add app/components/BlankPage.tsx out
git commit -m "まっさらページを更新"
git push origin main
```

Vercel が自動でデプロイするので、数分後に 4 つの URL で新しい内容が表示されます。

---

## 4 つそれぞれで「別の内容」にしたい場合

いまは **4 つとも同じ「まっさらなページ」** です。  
「ultra.omamori だけ違うデザインにしたい」といった場合は、**ドメインごとに中身を変える**必要があります。

**手順の例:**

1. **`app/components/BlankPage.tsx`** で、**表示中のドメイン** を参照する  
   - 例: `typeof window !== "undefined" && window.location.hostname` で `ultra.omamori.whenpigsfly.jp` などが取れる  
2. その値に応じて、表示する内容を切り替える  
   - 例: `if (hostname === "ultra.omamori.whenpigsfly.jp") return <UltraOmamoriPage />;` のように分岐  
3. 各サイト用のコンポーネント（例: `UltraOmamoriPage.tsx`）を用意して、中身を書く  

「まずは 4 つとも同じまっさらでよい」なら、**`BlankPage.tsx` を 1 つ編集するだけで OK** です。

---

## まとめ

- **まっさらなページの見た目を変える** → **`app/components/BlankPage.tsx`** を編集  
- **反映** → `npm run build` → `out` をコミット → `git push`  
- **4 つを別々のデザインにする** → `BlankPage.tsx` 内で `window.location.hostname` を見て出し分ける  

この手順で、4 つのサイトをまっさらなページのまま運用・変更できます。
