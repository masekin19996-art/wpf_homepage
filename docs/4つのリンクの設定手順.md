# 4つの角丸長方形のリンク先 — ここ見ればあとはやるだけ

1ページ目・上段の **クリックできる4つ** のリンク先は、次のホスト名で **すでに設定済み** です。

| 位置（上段・左→右） | ホスト名 | 設定済みURL |
|---------------------|----------|----------------------------------------|
| 1番目 | ultra.omamori | https://ultra.omamori.whenpigsfly.jp |
| 2番目 | snobbycover | https://snobbycover.whenpigsfly.jp |
| 3番目 | hickeytattoo | https://hickeytattoo.whenpigsfly.jp |
| 4番目 | gummygummy | https://gummygummy.whenpigsfly.jp |

---

## やること（この順だけ）

### 1. URLを変えたいときだけ

**編集するファイルは 1 つだけ:** `app/data/homeLinks.ts`

- 上段左から 1番＝配列の 1 つ目、2番＝2 つ目、3番＝3 つ目、4番＝4 つ目
- 各要素の **`url`** を変えれば、クリック時の飛び先が変わる
- **`title`** はボタン名（ツールチップ等）。必要ならここも変更

### 2. ビルド

```bash
npm run build
```

### 3. 本番に反映したいとき

```bash
git add app/data/homeLinks.ts out
git commit -m "4つのリンク先を更新"
git push origin main
```

（Vercel が自動でデプロイするので、あとは待つだけ）

---

## まとめ

- **リンク先の定義** → `app/data/homeLinks.ts` だけ見ればよい  
- **やること** → （必要なら URL を編集）→ `npm run build` → 本番なら `git add` / `commit` / `push`  
この手順だけ守れば、4つの角丸長方形のドメイン設定は完了です。
